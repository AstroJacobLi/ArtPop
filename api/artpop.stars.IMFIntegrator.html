IMFIntegrator¶

class artpop.stars.IMFIntegrator(params, m_min=0.1, m_max=120.0)¶

Bases: object

A helper class for numerically integrating the IMF.

Parameters¶

paramsstr or dict

Which IMF to use, if str then must be one of pre-defined: ‘kroupa’, ‘scalo’ or ‘salpeter’. Can also specify broken power law as dict, which must contain either ‘a’ as a Float (describing the slope of a single power law) or ‘a’ (a list with 4 elements describing the slopes of a broken power law) and ‘b’ (a list with 3 elements describing the locations of the breaks).

m_minfloat, optional

Minimum stellar mass.

m_maxfloat, optional

Maximum stellar mass.

weights(mass_grid, norm_type=None, norm_mass_min=None, norm_mass_max=None)¶

Calculate the weights of the IMF at grid of stellar masses.

The IMF function dn / dM is given by dn / dM = M**a0 / c0 for M < b0 dn / dM = M**a1 / c1 for b0 < M < b1 dn / dM = M**a2 / c2 for b1 < M < b2 dn / dM = M**a3 / c3 for b2 < M,

where c0, c1, c2, c3 are constants that are defined to ensure continuity at the breaks.

Parameters¶

mass_gridndarray

Stellar mass grid.

norm_typestr, optional

How to normalize the weights: by ‘number’, ‘mass’, or the ‘sum’.

norm_mass_minint or None, optional

Minimum mass to use for normalization. If None, use minimum of mass_grid will be used.

norm_mass_maxint or None, optional

Maximum mass to use for normalization. If None, use maximum of mass_grid will be used.

Returns¶

weightsndarray

The weights associated with each mass in the input mass_grid.
